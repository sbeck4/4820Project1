# How to Run
1. Find the ip address of the receiver computer
2. Assign that ip address to the "ip" variable in knobs.py
3. Run the video player (vertical screen) by typing kivy video.py into the command line 
4. Run the knobs file (horizontal screen) by typing kivy knobs.py into the command line
5. Place a tangible on the knobs on the horizontal screen to play a video on the vertical screen

# Platforms
- Any computer with python and kivy

# Tangibles
- Use with any three pronged tangible that creates a triangle on the screen
